# 🌍 PlanTripper – Smart Travel Planner (Frontend)

## 🚀 Overview

**PlanTripper** is a modern, interactive travel planning web application designed to simulate a real-world booking experience using only frontend technologies. It allows users to explore destinations, compare transport options, manage bookings, and personalize their travel—all within a responsive and visually engaging interface.

This project focuses on delivering a **complete user experience (UX)** by simulating backend functionality using `localStorage`.

---

## ✨ Features

### 🔍 Smart Search

* Autocomplete suggestions for cities and destinations
* Swap origin and destination
* Date selection with an interactive calendar

### 📅 Travel Planning

* Departure & return date selection
* Passenger management (adults, children, infants)

### 🚗 Transport Comparison

* Compare **Flight, Train, Bus, and Car**
* View price, duration, and comfort level
* Highlights the **best option based on budget**

### 🏝️ Destinations Explorer

* Beautiful destination cards with ratings and pricing
* Detailed modal view with:

  * Description
  * Best time to visit
  * Transport options

### ❤️ Wishlist

* Add/remove destinations
* Stored locally for quick access

### 📖 Booking System (Simulated)

* Book trips with selected options
* Ticket generation UI
* Booking history timeline

### 👤 Authentication (Simulated)

* Login / Register system using `localStorage`
* Profile management
* Preferences (budget, transport)

### 🌙 Dark Mode

* Toggle between light and dark themes
* User preference saved locally

### 🎯 Budget Filter

* Adjust travel budget dynamically
* Highlights matching destination options

## ⚙️ How It Works

* All application data (user, bookings, wishlist, preferences) is stored in **browser localStorage**
* No backend or database is used
* The app simulates real-world behavior like:

  * Authentication
  * Booking system
  * Data persistence
